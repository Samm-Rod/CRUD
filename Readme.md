## :dart: O que é isso ?
**Crud em TI é um sistema de registro de dados que se vincula a um banco de dados**
> **Você pode:**
> + Criar (Create);
> + Ver (Read);
> + Atualizar (Update);
> + Deletar (Delete).

## :moyai: O que ele faz ?
```
Como dito acima ele tem essas funcionalidade
porém, nesse caso ele inclui "nomes" e "cidade".
```
